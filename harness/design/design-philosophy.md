# Design Philosophy

PayOrbit는 금융 서비스이지만 전통적인 딱딱한 느낌보다
"신뢰감 있는 현대적 핀테크" 느낌을 지향한다.


디자인 목표:
- 복잡하지 않을 것
- 사용자가 긴장하지 않을 것
- 입력 흐름이 빠를 것
- 모바일 WebView 환경에서 답답하지 않을 것

지양:
- 과도한 그림자
- 무거운 카드 UI
- 너무 많은 색상 사용
- 복잡한 애니메이션
- 전형적인 관리자 페이지 느낌

지향:
- Toss / Linear / Stripe 스타일의 미니멀 UI
- 큰 타이포 중심
- 충분한 여백
- 단순한 액션 구조


절대 금지:
- bootstrap 느낌 UI
- 과도한 gradient
- glassmorphism 남용
- 작은 버튼
- cramped layout
- 지나치게 많은 divider
