# UI Style Guide

## Border Radius
- 기본: 12px
- 카드: 16px
- 버튼: 12px

## Shadow
- 매우 약하게 사용
- elevation 최소화

## Typography
- 헤드라인은 bold
- body는 regular
- 작은 회색 텍스트 남발 금지

## Spacing
- section 간격: 32px
- input 간격: 16px
- button top margin: 24px

## Buttons
- primary CTA는 화면당 1개만 강조
- secondary 버튼은 outline 사용
